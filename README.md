# 复杂系统的涌现：认知与计算的组织原理

## 仓库的基本介绍
WHAT：这个仓库详细记录我探索一个核心问题的过程：**认知系统（大脑/心智）和计算系统（AI/操作系统）等复杂系统背后相似的组织原理**。

WHY：我对**处理信息、实现功能的复杂系统（无论是生物的还是人工的）背后相似的组织模式以及其背后的数学直觉与哲学思想**着迷，希望吸引同路人并相互启发、纠偏。
